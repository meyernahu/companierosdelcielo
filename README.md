# Compañeros de Cielo 🕊️

**Compañeros de Cielo** es una pequeña aplicación web diseñada para ofrecer a los usuarios un espacio de inspiración diaria, permitiéndoles generar de forma aleatoria un Santo de la Iglesia Católica para conocer su vida, patronazgo y enseñanzas.

Este proyecto combina un Frontend ligero y rápido con un proceso de recolección de datos automatizado (Web Scraping) en el Backend.

## 🚀 Características

* **Generador Aleatorio:** Selecciona y muestra un santo distinto con cada interacción.
* **Diseño Responsivo:** Interfaz limpia, sin dependencias complejas, adaptable a dispositivos móviles y de escritorio.
* **Datos Automatizados:** Incluye un script en Python para extraer (hacer scraping) y actualizar la base de datos de santos en formato JSON desde fuentes católicas confiables.

## 🛠️ Tecnologías Utilizadas

* **Frontend:** HTML5, CSS3, Vanilla JavaScript.
* **Datos:** Archivo local `santos.json`.
* **Scraping / Backend:** Python 3 (librerías `requests` y `BeautifulSoup4`).
* **Despliegue:** GitHub Pages (Client-Side rendering).
